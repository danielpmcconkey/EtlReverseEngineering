  <action>
    Create src/workflow_engine/engine.py:

    class Engine:
        def __init__(self, config: EngineConfig):
            - Store config
            - Create RNG from config.seed (random.Random(seed) if seed, else None)
            - Build node registry via create_node_registry(rng)
            - Get structlog logger
            - Run validate_transition_table() at init -- raise if errors

        def run_job(self, job: JobState) -> JobState:
            - Bind logger with job_id=job.job_id
            - While job.status == "RUNNING":
              1. Get node from registry by job.current_node
              2. Execute node: outcome = node.execute(job)
              3. Resolve transition: next_node = TRANSITION_TABLE[(job.current_node, outcome)]
                 - If key missing, raise ValueError with descriptive message
              4. Log transition: event="transition", node=job.current_node, outcome=outcome.name, next_node=next_node, main_retry=job.main_retry_count, conditional_counts=dict(job.conditional_counts)
              5. If next_node == "COMPLETE": job.status = "COMPLETE"
                 Else: job.current_node = next_node
            - Log job complete: event="job_complete", job_id=job.job_id, final_status=job.status
            - Return job

        def run(self) -> list[JobState]:
            - configure_logging() at start
            - For i in range(config.n_jobs):
              - Create fresh JobState(job_id=f"job-{i+1:04d}")
              - Run job via run_job()
              - Collect results
            - Return list of completed JobStates

    IMPORTANT: Do NOT call configure_logging() in __init__. Call it in run() so tests can control logging setup independently. Tests that need to capture log output should configure structlog themselves with a capture processor.

    Write tests/test_engine.py with all behaviors above. For log capture tests, use structlog's testing utilities or capture stdout and parse JSON lines. The test for log_completeness should count lines where event=="transition" and verify exactly 27 per job.
